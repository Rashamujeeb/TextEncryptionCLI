# Text Encryption CLI 🔒

This is a Python CLI tool for encrypting and decrypting text using **AES, DES, and RSA** for secure data protection.

## 📌 Features:
- **AES Encryption (Advanced Encryption Standard)**
- **DES Encryption (Data Encryption Standard)**
- **RSA Encryption (Public-Private Key Encryption)**
- **User-friendly CLI for easy encryption & decryption**
- **Error handling for incorrect inputs**

## 🚀 How to Use:
1. Run the script:
2. Choose **Encryption (1) or Decryption (2)**.
3. Select the encryption algorithm:
- `1` for AES
- `2` for DES
- `3` for RSA
4. Enter the text to encrypt/decrypt.



## 🔧 Requirements:
Install dependencies using: 
pip install pycryptodome
